# Coding Compass Challenge Completions

[Coding Compass](https://coding-compass.123ishatest.com/) is a new approach to programming tutorials.
Instead of copying code you vaguely understand from a video, Coding Compass presents you with hand-crafted Challenges.
Each challenge points you in the direction of a specific architectural principle.

This repository is a collection of all challenge completions.
It is highly recommended to go into a challenge without looking at other peoples solutions, but make sure to take a look after you are done!

## Submit a solution

Open a pull request against the `main` branch with your solution to the challenge.
Please organize all your files into a single folder with your username, and make sure it is under the correct challenge

Feel free to provide an introduction to your solution, answering any or none of these questions:
- What architectural choices did you make and why?
- Did you run into any problems and had to adapt?
- Did you have fun?
